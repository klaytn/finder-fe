# Slush

## Introduction

KlaytnFinder UI library.

## Installation

1. Create a `.npmrc` file with the following content in the directory where you want to install:

    ```
    @klaytn:registry=https://npm.pkg.github.com/
    ```

2. Refer to the [GitHub guide](https://docs.github.com/en/packages/working-with-a-github-packages-registry/working-with-the-npm-registry#authenticating-to-github-packages) to log in to npm.
3. Run `npm install @klaytn/slush`
